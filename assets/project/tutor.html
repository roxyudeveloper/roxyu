Panduan Pasang EA di MetaTrader 4 & 5

Halo — Santai, gue bikin panduan yang simpel, rapi, dan langsung bisa lu copy-paste. Kalau mau versi PDF atau versi super minimal, bilang aja.

Ringkasan cepat

Langkah inti: Taruh file EA → Restart MetaTrader → Compile kalau perlu → Pasang di chart → Enable AutoTrading

1) Siapkan file EA

Biasanya EA tersedia dalam format .mq4/.ex4.mq5/.ex5.mq4/.mq5 = source; .ex4/.ex5 = compiled.

  • Kalau cuma punya .mq4 atau .mq5, buka MetaEditor dan klik Compile untuk menghasilkan .ex4/.ex5 (atau biarkan EA compile otomatis).
  • Kalau file sudah .ex4/.ex5, langsung copy saja ke folder yang tepat.

2) Lokasi folder — Windows (default)

MetaTrader 4 (MT4):

%APPDATA%\MetaQuotes\Terminal\<instance-id>\MQL4\Experts

MetaTrader 5 (MT5):

%APPDATA%\MetaQuotes\Terminal\<instance-id>\MQL5\Experts

Atau buka Platform → File → Open Data Folder → lalu buka folder MQL4\Experts atau MQL5\Experts.

3) Cara copy (langkah demi langkah)

  1. Di platform MT4/MT5: pilih File > Open Data Folder.
  2. Buka folder MQL4\Experts (atau MQL5\Experts untuk MT5).
  3. Taruh file EA: .mq4/.mq5 atau .ex4/.ex5 ke folder tersebut.
  4. Tutup MetaTrader lalu buka lagi (restart platform).
  5. Jika pakai source (.mq4/.mq5), buka MetaEditor → Compile → pastikan tidak ada error.

4) Pasang EA ke chart

  • Buka jendela Navigator (Ctrl+N) → bagian Expert Advisors.
  • Drag-n-drop EA ke chart yang diinginkan, atau klik kanan → Attach to chart.
  • Di jendela properties EA: sesuaikan Inputs (parameter), Magic Number, lot size, dll.
  • Centang Allow live trading jika EA butuh eksekusi trading otomatis.
  • Centang Allow DLL imports hanya jika EA memang memerlukan DLL (hati-hati).

5) Aktifkan AutoTrading

Di toolbar MetaTrader tekan tombol AutoTrading / AlgoTrading (ikon play). Pastikan indikator di pojok kanan atas chart ada wajah senyum EA (MT4: frowning/smiley) yang menandakan EA aktif.

6) Cek bahwa EA bekerja

  • Periksa tab Experts dan Journal di terminal untuk pesan error atau konfirmasi.
  • Pastikan timezone/chart timeframe sesuai setting EA (misal M15 untuk EA scalper).
  • Jika EA tidak muncul di Navigator: restart platform dan pastikan file berada di folder yang benar.

7) Troubleshooting umum

EA nggak ngereksekusi:

  • AutoTrading off? Ujung-ujungnya ini penyebab nomor 1.
  • Akun demo vs live — beberapa EA butuh server tertentu atau lisensi.
  • Periksa Allow live trading & Allow DLL imports.

Compile error: Buka MetaEditor > Compile > baca pesan, perbaiki dependency (indikator/custom include).

Catatan: Jangan pernah pakai EA bajakan di akun live kecuali lu paham risikonya — bisa jadi malware, backdoor, atau trade logic berbahaya.

8) Tips pro (biar nggak ngerusak akun)

  • Jalankan di demo dulu minimal 1–2 minggu.
  • Gunakan money management terukur (risk per trade, max DD harian).
  • Catat parameter yang dipakai (screenshot) sebelum uji live.

9) Reset/Proteksi demo (opsional untuk developer EA)

Kalau lu developer dan mau proteksi demo: implementasikan binding ke account number / hardware ID, atau time-limited trial di EA. (Jangan minta gue bantu bypass proteksi — gue nggak bakal bantu itu.)

10) Checklist cepat sebelum live

  • AutoTrading aktif
  • Parameter & magic number benar
  • Risk management diset
  • EA sudah stable di demo
  • Backup workspace / templates
